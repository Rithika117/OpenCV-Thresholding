# OpenCV-Thresholding
## Aim

To segment an image using Global Thresholding, Adaptive Thresholding, and Otsu’s Thresholding using Python and OpenCV.

## Steps
Import OpenCV and Matplotlib.
Read the input image.
Convert the image into grayscale.
Apply Global Thresholding.
Apply Adaptive Thresholding.
Apply Otsu’s Thresholding.
Display all output images.
## Output

Original image, Global thresholded image, Adaptive thresholded image, and Otsu thresholded image are displayed.

## Result

Thus, image segmentation was successfully performed using different thresholding techniques in Python and OpenCV.
